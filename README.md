# shadow-reframe

A Leiningen template for a shadow-cljs proejct with the following libraries:
 - re-frame & reagent
 - semantic-ui
 - less
 - dev-cards
 - 10x (debugging support)
 - circle-ci build files

## Usage

lein new shadow-reframe my-shiny-new-site
